🌐 ROX Globe — Interactive 3D World Map
ROX Globe is a sleek and interactive 3D globe built using HTML, CSS, and JavaScript. It visualizes the Earth with country and continent outlines using globe.gl and three.js, providing a modern and immersive experience right in the browser.

✨ Features
🌍 3D rotatable Earth with real-world country outlines

💡 Clean, responsive UI with glass-style title bar

⚡ Powered by globe.gl, three.js, and topojson

💻 No backend required — runs entirely in the browse
